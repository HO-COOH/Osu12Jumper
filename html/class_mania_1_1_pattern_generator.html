Finds a new column in which a HitObject can be placed.

Parameters
initialColumnThe initial column to test. This may be returned if it is already a valid column.
lowerBoundThe minimum column index. If null, randomStart is used.
upperBoundThe maximum column index. If null, totalColumns is used.
nextColumnA function to retrieve the next column. If null, a randomisation scheme will be used.
validatorA function to perform additional validation checks to determine if a column is a valid candidate for a HitObject.
patternsA list of patterns for which the validity of a column should be checked against.

A column is not a valid candidate if a HitObject occupies the same column in any of the patterns.

Returns
A column which has passed the validator check and for which there are no HitObjects in any of patterns occupying the same column.
Return values
-1If there are no valid candidate columns.
